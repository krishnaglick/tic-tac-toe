
# Tic Tac Toe

## Install
 - Run `git clone https://github.com/krishnaglick/tic-tac-toe.git`
 - `cd tic-tac-toe`
 - `yarn` or `npm i`

## Run

### Dev
 - In one terminal window: `yarn run build:watch` or `npm run build:watch`
 - In another, `yarn start` or `npm start`
 - Visit `http://localhost:9000`

### Prod
 - `npm run build`
 - `npm start`
